## 这是什么？

这是一个多语单词对照表，收录了 **罗曼语族、日耳曼语族、斯拉夫语族、希腊词根、拉丁短语、圣经词汇、神话词汇、文字编码术语** 共 8 大类语言数据。

- 每个单词都标注了它在**英语、法语、西班牙语、意大利语、葡萄牙语、德语、荷兰语、瑞典语、拉丁语、俄语、波兰语、希腊语**等多种语言中的对应形式
- 还包含词源解释、词根分析、短语用法等辅助信息
- 其中俄语和波兰语的翻译由 AI 生成，在页面上以特殊样式标记（橙色斜体背景），方便区分人工数据与 AI 数据

## 数据来源

本项目数据来自以下书籍和资源：

| 数据源 | 说明 |

| [《The Loom of Languages》](https://book.douban.com/subject/2174464/) | 语言学习经典著作，提供罗曼语族与日耳曼语族的对照单词表（3,372 条） |
| [《你应该记住的拉丁语1234句》](https://book.douban.com/subject/26882919/) | 拉丁语短语合集，提供常用拉丁短语 5,389 条 |

> ⚠️ 俄语和波兰语翻译由 AI 辅助生成，可能存在不准确之处，建议配合专业词典使用。

## 主要功能

### 切换不同语言族

在顶部标签栏选择想看的词汇分类：

| 标签 | 内容 |
|------|------|
| 全部 | 所有词汇 |
| 罗曼语族 | 法语、西班牙语、意大利语、葡萄牙语等 |
| 日耳曼语族 | 德语、荷兰语、瑞典语、英语等 |
| 斯拉夫语族 | 俄语、波兰语 |
| 希腊词根 | 希腊词根及英语派生词 |
| 拉丁短语 | 常用拉丁短语 |
| 借词 | 拉丁借词 |
| 神话 | 希腊罗马神话词条 |
| 圣经 | 圣经相关拉丁词汇 |
| 编码 | 文字编码术语 |

### 搜索

在搜索框中输入任意语言的词汇（或中文释义），即可实时过滤结果。

### 调整每页显示条数

右上角可以选择每页显示 **50 / 100 / 200** 条。

### 显示/隐藏任意列

每个标签页都有列名开关（位于第三行），可以自由选择显示或隐藏某些语言列，只留下你关心的语言。

### 查看完整释义

如果某个单元格的内容过长被截断，**点击该单元格**即可弹出完整内容。

### 发音示范

鼠标悬停在某些词汇上时，会出现一个小音符按钮，点击可以跳转到 Forvo 网站听发音。

### 切换暗色模式

点击右上角的月亮/太阳图标可以切换暗色模式。

## 常见问题

**Q: 需要联网吗？**
> 在线版本需要联网使用，但速度很快（数据已做压缩加载）。本地离线版本不需要网络。

**Q: 数据有多少条？**
> 约 10,000+ 条词汇记录，涵盖 12+ 种语言。

**Q: 俄语和波兰语的翻译准确吗？**

> 请尝试使用更新版本的 Chrome、Firefox、Safari 或 Edge。如果问题持续，请尝试清除浏览器缓存后重试。
